# Contributing to Literature Review Tools & Resources

Thanks for your interest in contributing!

## Guidelines

### Adding a new tool

1. Make sure the tool is relevant to academic literature review
2. Check that it's not already on the list
3. Add it in the appropriate category
4. Use this format:

```markdown
- **[Tool Name](https://url.com/)** â€” Brief description. Key feature.
```

Or for tables:

```markdown
| [Tool Name](https://url.com/) | Price | Best for description |
```

### Quality standards

- Tools should be actively maintained
- Prefer tools with free tiers or trials when possible
- Include both open-source and commercial options
- No affiliate links

### Pull request process

1. Fork the repository
2. Make your changes
3. Submit a pull request with a clear description
4. Wait for review

### Suggesting changes

If you're not comfortable with pull requests, [open an issue](https://github.com/joepacal/academic-resources/issues) describing what you'd like to add or change.
